 Στο txt αρχειο που θα φτιάξουμε πως θα λειτουργει; θα το παίρνουμε σαν παραμετρο μαζι με τα host που θα τρέξουμε; Επίσης στη γραμμή του κάθε ρούτερ ποιες ip θα έχουμε και για ποιο λόγο;

το configuration file θα υπάρχει μόνο στον host που έχει ρόλο client.
Σε κάθε γραμμή θα περιέχεται το όνομα κάθε host ή router ακολουθούμενο από την IP (σε περίπτωση host) ή τις IPs (όλων των διεπαφών σε περίπτωση router).
Η πληροφορία αυτή θα αξιοποιηθεί σε σημεία: αρχικά αν προσέξεις, όταν κληθεί το script του client ακολουθείται από μια λίστα με ονόματα hosts που θα θα χρησιμοποιηθούν ως servers οπότε η αντιστοίχηση ονομάτων των hosts σε IPs θα γίνει με βάση αυτό το αρχείο, το οποίο καλό είναι να συμπεριλάβετε σται παραδοτέα σας.
Επίσης, όταν ο client λάβει από τον κάθε server τα αποτελέσματα των μετρήσεων σχετικά με latency, number of hops και IP path για κάθε ζεύγος servers, θα πρέπει να αντιστοιχήσει το μονοπάτι σε επίπεδο IPs σε μονοπάτι σε επίπεδο routers. 
Οπότε γι αυτό το λόγο χρειάζονται οι IPs και από τους hosts και από τους routers.
Επίσης να υπενθυμίσω ότι τα scripts σας πρέπει να είναι δυναμικά και να μην έχουν κάποια 'καρφωτή' IP αλλά να διαβάζουν τις IPs που περιέχονται στο configuration file κάθε φορά.
