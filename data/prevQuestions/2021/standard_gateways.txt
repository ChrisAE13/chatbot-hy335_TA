με την εντολή "route add default gw IP_ADDRESS INTERFACE_NAME" έχουμε συνδέσει το interface του κάθε host με το διαδίκτυο; Επίσης γνωρίζει το switch σε ποιον router θα πάει ή πρέπει να το κάνουμε κάπως εμείς αυτό (και αν ναι με ποια εντολή;)

σχετικά με τον ρόλο της εντολής "add default gw" περιγράφεται λεπτομερώς στην σελίδα 3 του tutorial που υπάρχει στον φάκελο του Project.
Σε γενικές γραμμές, ο ορισμός του default gateway επιτρέπει την επικοινωνία του host με τις IPs εκτός subnet.
Απλά, για την ρύθμιση που θα κάνετε για τα gateways δώστε βάση στον παρακάτω περιορισμό που αναφέρει η εκφώνηση
" all hosts connected to CERN and EPFL use GENE as standard gateway and the hosts at ETHZ use ZURI"
Επίσης, δεν χρειάζεται για το τρέχον ερώτημα να κάνετε κάποια ρύθμιση στα switches.

Οπότε αν τρέξουμε απλά την εντολή αυτή δεν έχουμε πει κάπως τα IP των host να πηγαίνουν στα αντίστοιχα routers, πρέπει να κάνουμε και κάτι άλλο;
Εμείς απλά βάλαμε ως default gw τα IP των interface που βάλαμε σε κάθε host. Πως γίνεται να τεστάρουμε αν το κάναμε σωστά;


σε κάθε host θα ορίσετε σαν default gateway την IP που έχετε αναθέσει στο router (δείτε ξανά το slide 10 από το τελευταίο φροντιστήριο σχετικά με τον τρόπο που αναθέτουμε IP σε router interface). Δεν θα βάλετε σαν default gateway την IP του host.
Επίσης, το gateway που θα επιλέξετε στον κάθε host σχετίζεται και με τον 2ο περιορισμό που περιέγραψα στο προηγούμεμο post.
