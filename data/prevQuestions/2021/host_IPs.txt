Σε κάθε host κάνω assign block IPS ή μια που θα διαλέξω εγώ;
Επισης τρέχοντας το ping θα πρέπει να κάνω μόνο ένα jump όπως έχει οριστεί το interface?(Το δεύτερο ερώτημα είναι πιο πολύ για επιβεβαιωση.)

σε κάθε host πρέπει να αναθέσετε μια IP από ένα block IPs που έχετε στη διάθεσή σας σαν AS. Το subnet mask που ορίζετε σε κάθε host περιγράφει την μορφή που έχουν οι IPs στο ίδιο subnet αλλά κάθε host θα έχει μια και μόνο IP.
Σχετικά με το 2ο μέρος της ερώτησής σου, το ping ελέγχει connectivity. Τα hops φαίνονται μέσω της εντολής traceroute.
