στο 2.5 ερώτημα μας λετε να εκτελέσουμε measurements στο server και να στείλουμε τα αποτελέσματα στον client. Αυτό στην python πως ακριβώς το κάνουμε? Υπάρχει κάποια εντολή που θα πρεπει να γράψω στο script file του server ετσι ωστε να ξεκινήσει να κάνει τις μετρήσεις(οταν λάβει το κατάλληλο μήνυμα απο τον client) και μετά να τις στείλει?

όπως εξηγήθηκε και στο φροντιστήριο, ο client θα πρέπει να στείλει ένα μήνυμα στον server σχετικά με τις IPs για τις οποίες θα πρέπει να εκτελέσει μετρήσεις.
Μόλις λάβει το μήνυμα ο server από τον client, το server script σας θα πρέπει να τρέξει τις εντολές ping και traceroute μέσω python στον server. Υπάρχει τρόπος τον οποίο επίσης είπαμε στο φροντιστήριο ώστε να τρέξετε εντολές λειτουργικού μέσω python (σαν να τις πληκτρολογούσατε σε terminal) και μάλιστα σας παρέχει τη δυνατότητα να επεξεργαστείτε κατάλληλα το output κάθε εντολής και να απομονώσετε τις πληροφορίες που σας ενδιαφέρουν.
