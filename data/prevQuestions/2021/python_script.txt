μήπως θα μπορούσατε να μας πείτε πως μπορούμε να χρησιμοποιήσουμε το python script που μας έχετε δώσει για να εκτελέσουμε το looking glass μέσα στο AS μας; Πρέπει κάπως να το ανεβάσουμε εκεί ή μπορούμε κάπως αλλίως;

όπως αναφέρουμε και στην σελίδα 9 της εκφώνησης, θα χρειαστεί να έχετε εγκαταστήσει την Python στο λειτουργικό σύστημά σας (αν έχετε Windows).
Στα περισσότερα UNIX συστήματα υπάρχει ήδη εγκατεστημένη η Python. 
Μια τρίτη λύση αν έχετε Windows είναι να δημιουργήσετε ενα Linux virtual machine στα Windows και αν τρέξετε εκεί το script. 
Μετά την εγκατάσταση της Python θα πρέπει να εγκαταστήσετε την βιβλιοθήκη Psycopg. Παρατίθενται σχετικά links στην εκφώνηση.
Στη συνέχεια θα μπορείτε να εκτελέσετε κανονικά το script database-query.py από το δικό σας pc λειτουργώντας ως looking glass γράφοντας την εντολή 
python database-query.py ASN-router_name
όπου ASN και router-name το ΑΣ και το όνομα του router που θέλετε να εξετάσετε.
