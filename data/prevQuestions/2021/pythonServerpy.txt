έχω μια ερώτηση σχετικά με το πως τρέχουμε το server.py από κάποιον server. Θα χρειαστεί να βάλουμε την IP_address που θα ακούει και το port που θα ακούει σαν arguments; πχ python server.py X.103.0.1 50000 ή κάπως αλλιώς;

σχετικά με το ερώτημά σου, μπορείς, με βάση τα όσα απάντησα σε παρεμφερή ερώτηση στο forum, στο script του server να βρίσκεις την IP του host και να γίνεται εκεί bind το socket.
Για το port number μπορείς να θεωρήσεις το ίδιο για ολους τους hosts (πχ όλοι οι servers να ακούν στο port 50000 ), οπότε δεν επηρρεάζει σε κάτι τη ροή του προγράμματος σας.  
Συνεπώς δεν χρειάζεται να τα συμπεριλάβεις σαν command line arguments στην κληση του server script.
