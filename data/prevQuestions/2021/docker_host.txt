Θα ήθελα να ρωτήσω γιατί το docker δεν δινει την IP του host αν τρέξω σε Python 
myhostname=socket.gethostname()
socket.gethostbyname(myhostname )
Το τσέκαρα και στον δικό μου υπολογιστή και δίνει κανονικά την IP.Υπάρχει κάποια περίπτωση να μην ξέρει ποια IP να δώσει;
Τις εντολές τις βρήκα απο το documentation της βιβλιοθήκης!

η λύση αυτή δεν αποδίδει στους συγκεκριμένους dockers σε σύγκριση με έναν κανονικό υπολογιστή καθώς όπως είχαμε πει και στο πρώτο φροντιστήριο σχετικά με dockers, είναι μια lightweight έκδοση λειτουργικού εφοδιασμένο με τις βασικές εντολές.
Μπορείς εναλλακτικά να καλέσεις την εντολή που δίνει την IP ενός υπολογιστή μέσα από τo python script (άλλωστε το ίδιο θα κάνετε και για την κλήση των ping και traceroute ) και να κάνεις parse το output της εντολής βρίσκοντας εν τέλει την IP του host.
