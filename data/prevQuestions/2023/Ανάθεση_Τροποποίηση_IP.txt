Καλησπέρα,έχουμε πρσθέσει κάποια interface που προσπαθούμε να τα βγάλουμε και δεν μπορούμε να βρούμε κάποιον τρόπο. Μπορείτε να μας γράψετε την εντολή που πρέπει να χρησιμοποιήσουμε για να αφαιρέσουμε κάποιο interface;

Μια διευκρίνιση σχετικά με το πως να κάνετε assign IP διευθύνσεις.
Μπορείτε με 2 τρόπους να κάνετε assign μια ip σε κάποιον host.
1. μέσω "ip address add /subnet> dev   name>"
2. μέσω "ifconfig name> /subnet> .

Γενικά για εισαγωγή και διαγραφή ip διευθύνσεων εμείς προτείνουμε τον πρώτο τρόπο (που υπάρχει και στην εκφώνηση). Αυτό δεν σημαίνει ότι ο δεύτερος τρόπος είναι λάθος.
ΠΡΟΣΟΧΗ!!
Εάν αποφασίσετε να εισάγετε μια ip με τον ένα τρόπο, θα πρέπει αν θέλετε να την αφαιρέσετε, να το κάνετε με τον αντίστοιχο, διαφορετικά θα υπάρχουν προβλήματα.
Για παράδειγμα, όταν κάνω assign μια ip με την εντολή ifconfig, εάν χρειαστεί να την αφαιρέσω, θα πρέπει να χρησιμοποιήσω την αντίστοιχη της ifconfig και όχι την ip address del.
Επίσης, αν κάποια ομάδα έχει κάνει ανάθεση μιας IP σε κάποιο interface αλλά δεν έχει ορίσει μάσκα, για να ορίσει την μάσκα θα πρέπει πρώτα να αναιρέσει την IP και μετά να αναθέσει εκ νέου IP και μάσκα αλλιώς θα υπάρχουν 2 interface με ίδιο όνομα και άλλο configuration. Αν θέλετε να διαγράψετε τα επιπλέον interfaces που προσθέσατε (οχι αυτά που είχε ο router by default και είναι up) γράφοντας no interface interface_name, όπου interface_name το όνομα του interface που θέλετε να αφαιρέσετε.
