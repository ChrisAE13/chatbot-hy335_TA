Υπάρχει περίπτωση να μπορώ να κάνω unset κάποιο route-map?
Ακολούθησα την διαδικασία που τα έκανα set βάζοντας no μπροστά απο κάθε εντολή αλλά παρόλα αυτά δεν αλλάζει κάτι.

σχετικά με το ερώτημά σου, για να πετύχεις αναίρεση ενός route-map, σωστά πληκτρολογείς no πριν από κάθε εντολή μετά όμως από την είσοδο σε configuration mode μέσω της εντολής conf t στο σωστό router.
Για παράδειγμα, αναφερόμενος στο screenshot της σελίδας 21 του Exercise Session 8, για κάθε μια εντολή που υπάρχει μπορείς να την αναιρέσεις πληκτρολογώντας no και μετά ολόκληρη την εντολή.
Τέλος πως διαπίστωσες πως δεν άλλαξε κάτι? στο συγκεκριμένο παράδειγμα θα πρέπει να χρησιμοποιήσεις την εντολή show run μετά το exit.

εμπαινα κανονικα σε conf t τοσο για τα route-maps οσο και σε router bgp για τα neighbors . Απλα μετά κάνοντας show ip bgp σε εναν router βλεπω παντου τα preferences 100 που συμφωνα με το φροντιστήριο θα πρεπει να υπαρχουν τοσο 100 οσο 50 και 20
(εβαλα αυτες τις τιμες στο δικο μου set )

αρχικά βάσει του slide 7 του Exercise Session 9, η τιμή 100 είναι η default τιμή για LOCAL-PREF. Επίσης, να υπενθυμίσω πως επιλέγονται prefixes με υψηλότερο LOCAL-PREF.
Γενικά οι τιμές που βλέπεις δεν προκύπτουν αποκλειστικά από το configuration που εφαρμόζεις σε έναν router αλλά και από το configuration που έχεις κάνει και στους υπόλοιπους routers εντός του AS σου. Με άλλα λόγια, δεν θα φαίνονται οι τιμές που θέτεις αν υπάρχουν prefixes με υψηλότερο LOCAL-PREF λόγω configuration σε άλλους routers εντός του AS σου.

Συμφωνώ παρόλα αυτά είναι λογικό να είναι όλα 100άρια;

αυτό που μπορώ να πώ είναι πως δεν είναι και απίθανο να έχουν την τιμή 100 αν τυχαίνει αυτή να είναι η μέγιστη τιμή LOCAL-PREF βάσει του configuration σου συνολικά εντός του AS.
