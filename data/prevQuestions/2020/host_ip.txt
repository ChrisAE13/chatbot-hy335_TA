Έχουμε ένα θέμα με την εντολή που δίνει το IP του Host.
 
Ενώ η εντολή socket.gethostname() επιστρέφει σωστά το όνομα,  η εντολή
 
socket.gethostbyname(hostname) αποτυχάνει.  Temporary failure in name resolution 
 
Έχουμε κάνει import τις βιβλιοθήκες που απαιτούνται, και δεν βρίσκουμε που είναι το πρόβλημα.
 
Σαν λύση, έχουμε βάλει τις IP σε ένα πίνακα, και τις βρισκουμε μέσω του host name. 
 
Είναι αποδεκτή λύση; Μήπως υπάρχει άλλη εντολή για να παίρνουμε τις Ip του host ;
υπάρχουν και άλλοι τρόποι για να βρεις την IP. Για παράδειγμα μπορείς να τρέξεις μέσα από Python την εντολή ifconfig και με κατάλληλη επεξεργασία του output να απομονώσεις την IP.
