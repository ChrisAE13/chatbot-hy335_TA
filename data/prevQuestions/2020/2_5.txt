Από την εκφώνηση της 2.5 καταλαβαίνουμε ότι θα πρέπει να έχουμε τα αρχεία client.py και server.py σε κάθε host, και να τα τρέχουμε ανάλογα. 
Using ‘scp’ command in the proxy, transfer your file from proxy to the host (or from
host to proxy) . 
πχ για καποιο host:  hostscp -r project root@16.106.0.1
Με αυτές τις οδηγίες μεταφέρουμε τα αρχεία στα hosts, αλλά δεν μπορούμε να τρέξουμε τα αρχεία της python.   "Python command not found"
Ti πρέπει να κάνουμε για να τρέξουμε αρχείο python ; 

αρχικά θα πρέπει να μεταφέρετε το python script για τον client σε έναν host όπως επίσης και το script για τον server σε έναν ικανό αριθμό από hosts που λειτουργούν ως servers (π.χ 3).
Για την μεταφορά αυτή, όπως εξηγούμε και στα slides αλλά και στο τελευταίο φροντιστήριο, αρχικά θα μεταφέρετε το αρχείο μέσω ενός FTP client (πχ FileZilla) στον proxy και από εκει θα χρησιμοποιήσετε την εντολή scp για μεταφορά προς τον host που θέλετε με βάση το syntax που περιγράφεται στα slides. Τα scripts πρέπει να μεταφερθούν και να εκτελεστούν στους αντίστοιχους hosts όχι στον proxy.  
Λογικά, πήγες να τρέξεις τα scripts στον proxy γι αυτό και εμφανίστηκε το μήνυμα λάθους που είπες.
