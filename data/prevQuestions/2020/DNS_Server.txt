Για να επιβεβαιώσουμε ότι ένας host μπορεί να χρησιμοποιήσει τον DNS server αρκεί να κάνει traceroute με ένα όνομα τύπου ATH-lond.group19 αντί μίας IP διεύθυνσης; Γιατί ο ίδιος τρόπος δουλεύει και στους routers ενώ σύμφωνα με την  εκφώνηση αυτό δεν είναι δυνατόν να συμβαίνει.


Ένας host επιβεβαιώνεις ότι χρησιμοποιεί τον DNS server όταν στο αποτέλεσμα του traceroute δίπλα από κάθε ip βλεπεις ενα όνομα τύπου (source-destination.groupX).
Οι routers δεν μπορούν να κάνουν traceroute με την επιλογη --resolve-hostnames (το επιβεβαιώνουμε γράφοντας την εντολή "traceroute ?" σε οποιονδήποτε router).
Γι'αυτό το λόγο προτιμάμαι να κάνουμε traceroute απο τους host.
