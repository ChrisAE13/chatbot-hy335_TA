όταν πάω να τρέξω το server.py από έναν host μου εμφανίζει ένα error στην συνάρτηση που παίρνει την IP του host. Έψαξα να δω αν δεν υπάρχει επικοινωνία με τον DNS server αλλά στέλνει κανονικά ping. 

ο τρόπος μπου αναφέρεις με την συνάρτηση είναι ένας τρόπος να βρεις την IP αλλά δεν είναι ο μοναδικός.
Μπορείς να τρέξεις την εντολή που έχετε δει στα φροντιστήρια για να βρίσκεται την IP με την διοαφορά ότι θα πρέπει να την τρ΄ξεις μέσα από το python script  (και όχι command line) και να κάνετε parse το output της εντολής ώστε να βρείτε την IP  του server host.
